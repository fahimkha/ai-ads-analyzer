# AI Ads Analyzer 
AI Ads Analyzer is a web-based tool that analyzes Meta Ads data. 
It predicts ad performance (Winner / Average / Loser) and provides recommendations. 
## Installation 
1. Clone the repo: 
git clone https://github.com/USERNAME/ai-ads-analyzer.git 
cd ai-ads-analyzer 
2. Install dependencies: 
pip install -r requirements.txt 
3. Run server: 
py -m uvicorn main:app --reload 
Open http://127.0.0.1:8000 in browser.
